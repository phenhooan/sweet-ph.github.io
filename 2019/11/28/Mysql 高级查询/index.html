Mysql 高级查询

SQL子查询

  1. SELECT使用嵌套子查询

  1. FROM使用嵌套子查询

  1. WHERE使用嵌套子查询

  1. SOME 与 ANG 于 EXIST使用


函数使用

  1. 平均服务期限 AVG(TIMESTAMPDIFF(MONTH, e.hiredate, ‘2019-11-28’))
  2. IFNULL(e.comm,0) 这个函数的作用是如果e.comm字段的为null，则将其转换为0

SOME,ANY,IN,ALL,EXISTS

  1. SOME,ANY 外部字段 至少大于该集合中一个字段，与逻辑运算符结合使用
  2. IN 外部字段 是否存在于字段集合当中，相当于 = SOME 或 = ANY
  3. ALL 外部字段 必须大于该集合中所有字段，与逻辑运算符结合使用
  4. EXISTS 只与子查询相关，即，子查询是否返回字段，对应TRUE与FALSE

练习题

  1. 建表
  • dept部门表(deptno 部门编号 / dname 部门名称 / loc 地点)

  • emp员工表(
    empno员工号/ename员工姓名/job工作/mgr上级人员编号/
    hiredate受雇日期/sal薪金/comm佣金/deptno部门编号)

  1. 解题

(1)．列出至少有一个员工的所有部门。


(2)．列出薪金比“SMITH”多的所有员工。


(3)．列出所有员工的姓名及其直接上级的姓名。


(4)．列出受雇日期早于其直接上级的所有员工。(同上,日期可直接拿来比较)


(5)．列出所有工作为“CLERK”（办事员）的姓名及其部门名称。

(6)．列出在部门“SALES”（销售部）工作的员工的姓名，假定不知道销售部的部门编号。


(7)．列出薪金高于公司平均薪金的所有员工。(反复查自己)


(8)．列出与“James”从事相同工作的所有员工。(排除自己)


(9)．列出薪金等于部门30中员工的薪金的所有员工的姓名和薪金。


(10)．列出薪金高于在部门30工作的所有员工的薪金的员工姓名和薪金。


(11)．列出在每个部门工作的员工数量、平均工资和平均服务期限。


(12)．列出所有员工的姓名、部门名称和工资。
