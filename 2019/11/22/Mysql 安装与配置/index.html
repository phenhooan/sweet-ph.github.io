Mysql 安装与配置

发表于 2019-11-22

下载解压版

官网下载网址:https://dev.mysql.com/downloads/mysql/

解压

选择一个路径解压压缩包 如: d:\mysql

安装服务

以管理员的方式打开cmd, 切换到安装目录下bin文件夹 。 d:\mysql\mysql-8.0.18-win64\bin

配置mysql

输入命令mysqld –initialize，回车执行命令后需要等待一小会儿，这个过程会在MYSQL的解压目录mysql-5.7.21-winx64下生成一个data的文件夹，里面有一个后缀名为.err结尾的文件

该文件里会有一个默认的初始密码
